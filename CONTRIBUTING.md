# Contribuir

Gracias por querer contribuir. Sugerencias rápidas:

- Haz un fork y crea una rama con un nombre descriptivo.
- Añade pruebas cuando añadas funcionalidad.
- Sigue las reglas de estilo del proyecto (usa `npm run lint` si existe).
- Abre un Pull Request indicando el objetivo y pasos para reproducir.

Para cambios grandes, abre primero un issue para discutir la propuesta.
